Raster

Raster is a Bootstrap default gird, but this gird has no gutter(margin/padding) and can be used with borders(lines). The borders are seperate elements, this way you can build your own raster. The raster class works on children grid elements, this means that you can use the default grid inside the raster. The lines have 2 classes, one for the position and the other for the responsive part(xs,sm,md,lg) to make sure it will match the bootstrap grid.
