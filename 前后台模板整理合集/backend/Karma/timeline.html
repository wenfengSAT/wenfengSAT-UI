Arrow connector

Timeline specs/features
  • 4 types of connectors(dot, arrow, label and circle)
    • all connectors(inc line) can be positioned at the top, middle and bottom of the event
    • use a connector line between events and connectors
    • use a mini label(this example = time)
  • use the footer instead of the header
  • remove header and footer
  • add bootstrap buttons inside the header/footer
  • use almost every kind of content inside the event
  • use bootstrap grid to build a layout inside the event
  • use a meta title/text below each event(works only with connector position top)

This footer can be removed!

You can flip me, and use the footer as an header! This way you can build a creative timeline.

Notice: only top connectors can have a meta event title(the text below the event box).
